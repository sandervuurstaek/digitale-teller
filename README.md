# Digitale teller
Met dit Excel-bestand kan het optimale installatiejaar van de digitale teller berekend worden voor zonnepaneleneigenaars aan de hand van een simulatie.

## Gebruik
1) Download het Excel-bestand.
2) Geef de gegevens van uw installatie in. Hier zijn al gegevens ingevuld van een voorbeeldinstallatie.
3) Bekijk wanneer u de digitale teller het best kan installeren aan de hand van de grafiek.

## Opmerkingen
- Let goed op de eenheden, gegevens die in de verkeerde eenheid staan zullen een foutief resultaat geven.
